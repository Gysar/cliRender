# cliRender
renders .obj files on the command line.

Just a little project to fight the boredom. 

Usage:

cmdrender.exe -w 400 -h 400 -f example.obj
cmdrender.exe -f example.obj
cmdrender.exe -w 100 -h 100

If no obj is provided, a cube is displayed. 

Right now, everything runs on the CPU and you have to change the code to change the rotation. Translation isn't implemented at all.

I only tested it on windows.
